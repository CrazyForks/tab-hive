# Chrome 插件安装指南

## 📥 下载插件

1. 下载 `0.1.2_0.zip` 文件到本地
2. 解压缩该文件到一个文件夹（例如：`iframe-all-extension`）

## 🔧 安装步骤

### 方法一：Chrome 浏览器

1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/` 并回车
3. 在右上角打开"开发者模式"开关
4. 点击"加载已解压的扩展程序"按钮
5. 选择刚才解压的文件夹
6. 安装完成！

### 方法二：Edge 浏览器

1. 打开 Edge 浏览器
2. 在地址栏输入 `edge://extensions/` 并回车
3. 在左下角打开"开发人员模式"开关
4. 点击"加载解压缩的扩展"按钮
5. 选择刚才解压的文件夹
6. 安装完成！

### 方法三：其他 Chromium 内核浏览器

大多数基于 Chromium 的浏览器（如 Opera、Brave 等）都支持类似的安装方式：

1. 进入浏览器的扩展程序页面
2. 开启开发者模式
3. 加载解压的扩展文件夹

## ✅ 验证安装

安装成功后，您应该能在浏览器的扩展程序列表中看到 "Tab Hive" 或 "iframe-all" 插件。

## 🚀 使用

安装完成后，点击插件图标即可打开 Tab Hive 应用，开始管理您的多窗口浏览体验！

## ❓ 常见问题

### Q: 为什么需要开启开发者模式？
A: 因为这是直接从文件加载的插件，不是从 Chrome 网上应用店安装的。开启开发者模式后才能加载本地的扩展程序。

### Q: 插件会自动更新吗？
A: 不会。需要手动下载新版本并重新加载。

### Q: 是否安全？
A: 完全开源，您可以在安装前检查源代码。插件不会收集任何个人信息。

## 📝 注意事项

- 每次浏览器更新后，可能需要重新确认加载该扩展
- 请不要删除解压后的文件夹，否则插件将无法正常工作
- 建议将解压的文件夹放在一个固定的位置
