📱 快速开始

  • 添加网站：点击空白格子，输入网站名称和网址（支持直接输入域名，如 google.com）

  • 调整布局：使用顶部的横向/竖向按钮调整网格数量
  • 拖放添加：直接从浏览器书签栏拖放网址到格子中
  • URL 导入：通过 URL 参数直接导入网站集合（详见下方说明）

🔄 多布局管理

Tab Hive 支持创建多个布局配置，方便你在不同工作场景间快速切换。

如何使用多布局？

  • 查看当前布局：顶部配置栏左侧显示当前布局名称
  • 切换布局：点击布局按钮打开下拉菜单，选择不同布局
  • 新建布局：点击下拉菜单中的 ➕ 按钮，输入布局名称（如"工作"、"学习"、"娱乐"）
  • 重命名布局：点击布局列表中的编辑图标，修改布局名称
  • 删除布局：点击布局列表中的删除图标（至少保留一个布局）

多布局使用场景

💼 工作模式

邮箱、项目管理、文档、监控面板

📚 学习模式

在线课程、笔记、文档、视频教程

🎮 娱乐模式

视频网站、社交媒体、新闻资讯

📊 监控模式

多个监控仪表盘、数据大屏

提示：每个布局都有独立的网站列表和行列配置，切换布局时会立即加载对应的所有网站。所有更改都会自动保存。

🔗 URL 参数导入

Tab Hive 支持通过 URL 参数直接导入网站集合，非常适合分享工作空间或快速设置。

使用方法

  • 简单格式：?urls=https://google.com,https://bing.com
  • JSON 格式：?urls=["https://google.com","https://bing.com"]
  • 完整配置：?urls=[{"url":"https://google.com","title":"谷歌"}]

可选参数

  • layoutName - 指定布局名称
  • rows / cols - 指定行列数
  • clearParams - 是否清除 URL 参数（默认 true）

完整示例

示例：创建一个社交媒体布局

http://localhost:3000/?urls=https://twitter.com,https://facebook.com,https://instagram.com&layoutName=社交媒体&rows=2&cols=2

🌐 布局分享功能

Tab Hive 支持布局分享，你可以分享自己的布局给其他用户，也可以浏览和使用他人分享的布局。

分享布局

  • 分享方式：在布局下拉菜单中，点击布局旁边的分享按钮（绿色图标）
  • 分享限制：每个 IP 每天最多分享 10 个布局
  • 自动保存：分享的布局会自动保存到本地

浏览共享布局

  • 切换标签：在布局下拉菜单中，切换到"共享布局"标签页
  • 搜索功能：使用搜索框快速查找感兴趣的布局
  • 导入方式：支持实时同步导入和拷贝导入两种方式
  • 浏览次数：显示每个布局的浏览次数统计

导入模式选择

🔗 实时同步导入

保持与原模板同步，当作者更新模板时自动更新

📋 拷贝导入

创建一个独立的副本，可以自由修改

注意：实时同步的布局如果被修改，会自动断开同步链接，变为独立副本。

❓ 为什么有些网站打不开？

原因：部分网站设置了 X-Frame-Options 安全策略，禁止在 iframe 中显示。这是为了防止点击劫持攻击。

解决方法：

  1. ✅ 最佳方案：使用桌面客户端（完全无限制）
  2. ⚡ 推荐方案：安装 Chrome 插件（大部分网站可用）
  3. 🔄 替代方案：使用其他不受限的网站
⬇️ 下载桌面客户端 🔌 下载 Chrome 插件

🔧 常见问题

Q: 网页版、Chrome 插件和桌面客户端有什么区别？

A: 网页版无需安装但受 CORS 限制；Chrome 插件简单安装，大部分网站可用；桌面客户端完全无限制，可加载任何网站。

Q: 如何通过 URL 参数导入网站？

A: 在 URL 后添加 ?urls=网站1,网站2 或使用 JSON 格式 ?urls=["网站1","网站2"]，支持指定布局名称和行列数。

Q: 布局分享功能如何使用？

A: 在布局下拉菜单中点击分享按钮（绿色图标）分享布局，切换到"共享布局"标签页浏览他人分享的布局，支持实时同步和拷贝导入两种模式。

Q: 实时同步和拷贝导入有什么区别？

A: 实时同步会保持与原模板同步，当作者更新时自动更新；拷贝导入创建独立副本，可以自由修改。修改实时同步的布局会自动断开同步链接。

Q: 配置会丢失吗？

A: 不会。所有配置都保存在浏览器的 localStorage 中（网页版）或应用数据目录中（客户端），会自动恢复。

Q: 支持哪些平台？

A: 桌面客户端支持 Windows、macOS 和 Linux 三大平台，支持 x64 和 ARM64 架构。

Q: 如何清除所有配置？

A: 点击顶部的"清除配置"按钮，确认后会删除所有布局和网站设置，并重置为默认状态。

Q: 如何在多个布局之间快速切换？

A: 点击顶部的布局按钮（显示当前布局名称），在下拉菜单中选择要切换的布局。建议为每个布局起一个容易识别的名称。

Q: 删除布局会丢失数据吗？

A: 是的，删除布局会永久删除该布局中的所有网站配置，且无法恢复。系统会要求确认操作，并至少保留一个布局。
